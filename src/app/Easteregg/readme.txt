Explications :
Pour activer l'easterEgg, allez sur la page d'accueil
et appuyez sur les touches "m", "a", "s", l'une après
l'autre. "MAS", c'est l'inverse de SAM, l'easterEgg  
donne l'impression d'être sous l'effet de l'alcool.
_____________________________________________________

Comment ça marche ?
easterEgg.js est une micro librairie conçue pendant
la nuit de l'info 2017 à Strasbourg.

Nous avons pensé cette librairie js dans le but de
pouvoir continuer à la développer dans l'avenir.

Pris par le temps nous avons dû faire usage de Jquery
pour certaine actions, cette dépendance disparaîtra
dans les versions suivantes.
_____________________________________________________

La suite :
Nous pensons continuer à développer cette librairie
et y ajouter plusieurs types de détections et plus de
types d'effets pas forcéments associés au thème de la
sécurité routière.
